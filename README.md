게시글 작성시 사진은 1개만 올릴 수 있고, 태그는 갯수 제한 없습니다.
대신 태그는 하나의 태그가 상품 선택이 완료되어야 다른 태그를 등록할 수 있습니다. 태그 창은 마우스만 올리면 나옵니다.
